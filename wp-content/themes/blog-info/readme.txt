== Description ==
Blog info is a new WordPress  theme that allows you to create  blogs and websites.The theme is well designed and does not require any other hard work to get it up and running right away. All the hard work has been done in this theme, it is clean and efficient.Theme features using the built-in WordPress Customizer.

== Frequently Asked Questions ==

== Installation ==

1. In your site's admin panel, go to Appearance > Themes and click `Add New`.
2. Type "Blog Info" in the search field.
3. Click `Install` and then `Activate` to start using the theme.
