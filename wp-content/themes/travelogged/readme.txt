travelogged is a complete travel blog based WordPress theme.

== Description ==
travelogged is a free WordPress theme suitable for travel blogs. The theme is a very simple and easy to use. You can just go to its customizer option settings and change its header settings, footer, sidebar, and other section of the theme cozily.

= How to Install Theme =

1. In your admin panel, go to Appearance > Themes and click the Add New button and Search for travelogged, OR
2. If you have theme zip file, click Upload and choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

= How to change content or customize site? =
1. In the admin area, go to Appearance > Customize
2. You wll find different options in customizer
2. Add/Edit Widgets, Page and Post
